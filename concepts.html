Advizmo Design System
5 Overview Screen Concepts
Each concept represents a different UX philosophy for the Advizmo Overview screen — transforming it from a dashboard into a Financial Operating System.
01
UX Philosophy: Conversational
Financial Advisor
The screen feels like a daily financial briefing prepared by a personal advisor. Warm, conversational, prioritized.
02
UX Philosophy: Systems
Financial Operating System
The screen behaves like an operating system. Recommendations become the primary experience. Everything else supports.
03
UX Philosophy: Guided Action
Today's Financial Plan
The screen focuses on what users should accomplish today. Recommendations become a guided plan with visible progress.
04
UX Philosophy: Goal-Oriented
Financial Health Dashboard
The hero focuses on financial health. Recommendations become ways to improve the user's score. Goal-oriented.
05
UX Philosophy: Optimization
Money Optimization Center
Every section answers "What should happen to my money next?" Focuses on optimization opportunities over balances.
