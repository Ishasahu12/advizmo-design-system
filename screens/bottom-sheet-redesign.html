Recommendation

Recommended Concept

Concept 1 (Guided Action List) is recommended as the primary design. It follows Apple HIG closely, is fast to scan, and scales well.

Why Concept 1 Wins

  • ✓ Follows Apple Settings pattern — users already know how to use it
  • ✓ Grouped sections reduce cognitive load
  • ✓ Each row has icon + title + description + chevron — complete information
  • ✓ Scales well with more or fewer actions
  • ✓ Premium feel through clean typography and spacing

When to Use Each Concept

Guided Action List
Default choice. Best for most users. Familiar pattern.
Financial Categories
When many actions exist. Category grid helps navigation.
Large Action Cards
For premium feel. Focus on one primary action.
Progressive Disclosure
When decision fatigue is a concern. Show less, reveal more.
Financial Assistant
For premium users who prefer conversational guidance. Highest trust feel.
