Touch feedback, transitions, and animation specs for each component.

State Card tap → expand details

Action Reject → red highlight, card removes

Feedback Haptic on accept/reject (light impact)

Card removal: 350ms slide + fade

Feedback Success: confetti micro-animation on harvest
Transitions
Enable: 400ms spring animation

Steps reveal: 200ms staggered fade
Completion: 500ms scale + checkmark draw

Accessibility Notes

WCAG 2.1 AA compliance requirements for each component.

Color Contrast
Red text on white: 5.74:1 ✓
Green text on white: 4.63:1 ✓
Amber text on white: 4.12:1 ✓
Blue text on white: 4.56:1 ✓
Screen Reader
• All cards: role="article" with aria-label
• Pills: aria-label describes state fully
• Buttons: descriptive aria-label (not just "Review")
• Progress: aria-valuenow, aria-valuemin, aria-valuemax
• Timeline: aria-label="Change history, X items"
